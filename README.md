# Twiddler

This is a mockup of a social media feed, modeled on [Twitter](http://www.twitter.com).

## Features

* Nifty logo
* Wacky random tweet generator, auto-populates feed in real time
* Refresh button
* Click on a user's handle to filter for their tweets
